Praxis der Computersysteme

Zum Betriebsystem Linux sind Unmengen Dokumentation im Internet frei verfügbar. Ein guter Startpunkt um sich ein bißchen umzusehen ist etwa die Linux homepage unter http://www.linux.org oder das Linux Dokumenation Project (LDP) etwa unter http://gd.tuwien.ac.at/opsys/linux/LDP/. Grob gliedert sich die Linux online Dokumenation in guides, HOWTOs, man pages, mailing lists, news groups,...

  • Guides ( http://gd.tuwien.ac.at/opsys/linux/LDP/docs.html#guide) sind umfaßende buchartige Dokumenent zu allgemeinen Themen wie Einstieg, Installation, Programmieren unter Linux, Linux Adminsitration, Netzwerk,... Besonders empfehlenswert für Linux-Neulinge ist der Linux Users' Guide von Larry Greenfield.
  • HOWTOs ( http://gd.tuwien.ac.at/opsys/linux/LDP/HOWTO/HOWTO-INDEX-3.html#ss3.1) sind längere Abhandlungen zu spezifischen Themen wie etwa Modeminstallation, Laptop-spezifische Konfiguration, Konfiguration für SCSI Controller, Busmäuse,... Besonders interessant für DOS/Win Erfahrene ist das DOS-Win-to-Linux-HOWTO ( http://gd.tuwien.ac.at/opsys/linux/LDP/HOWTO/DOS-Win-to-Linux-HOWTO.html).
  • Zu jedem Linux (Unix) Befehl gibt es eine detaillierte Beschreibungen; die man (manual) page. Sie ist mittels der Kommandozeileneingabe: man befehl auf jedem Linux (Unix) system abrufbar.

Natürlich gibt es auch Bücher zum Thema Linux. Eine Zusammenstellung findet sich z.B. im Reading-List-HOWTO ( http://gd.tuwien.ac.at/opsys/linux/LDP/HOWTO/Reading-List-HOWTO.html) oder auf der linux homepage. Eine lange Liste von Grudlagenbüchern findet sich unter dem url http://www.linux.org/books/basic.html.

In der Vorlesung werde ich mich stark an "Running Linux" von Matt Welsh und Lar Kaufman orientieren.

Roland Steinbauer, 12.10.1999.
